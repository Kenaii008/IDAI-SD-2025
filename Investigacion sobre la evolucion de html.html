HTML,¿Que es? ¿Como se usa?

HTML son las siglas de Hiper Text Markup Language es más que un lenguaje marcado de hipertexto.

Importante definir que no es un lenguaje de programación.

HTML 5 va a describir lo que es el contenido y la estructura de mi página web.

Todo lo que se observa dentro de la página, ya sean textos, enlaces, imágenes, todo se va a describir con HTML 5 y también cómo va a estar maqueta y estructurada.

DOCTYPE

La primera línea en leer los navegadores es el tipo de documento, de esta forma ya está preparado para adaptarse al tipo de documento.
En caso de no indicarlo el navegador entrará en Quirk mode, donde activa un modo de retro compatibilidad con páginas antiguas, lo que procesará de forma diferente etiquetas HTML o propiedades CSS

HEAD

Title establece el titulo de la pagina
Link establece una relacion con otro documento externo.
Meta estabece un metadato.
Style permite indicar estilos CSS.
Script indica un script a cargar
